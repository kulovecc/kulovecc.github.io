未成年、裸聊、暗网，起底招募童星背后的隐秘!

Posted on 2020-03-14 Disqus:

之前在知乎回答过一些关于社会工程学的问题，所以会不时收到一些请求帮助的私信，大多数私信无非是被骗、出轨，让帮忙查一些信息，我对这些是比较嗤之以鼻的，一来对这些事情不感兴趣，二来是最近比较忙，没有精力。直到有一天…

事件起因

大概半个月前，我收到一条私信，是一位小姑娘发来的，内容大概是说：她在网上看到了招募童星的帖子，很感兴趣就联系了所谓的星探，但后来却一步步落入对方的圈套，不仅被骗了钱，甚至还被拍裸照威胁。

为了让更多人了解网络招募童星的骗局以及保护隐私的重要性，我决定把这件事情通过另一种角度尽可能真实地还原出来。至于真假，全凭各位看官分辨。

可以看得出她很无助，能怎么办呢？告诉家长？报警？最后会不会闹得满城皆知？我不知道。但这种事情对一位还未成年的小姑娘来说打击无疑是巨大的，所以我决定尽可能地帮助她：将被拍摄的照片删掉；把小姑娘的压岁钱追回来。随后我让她加了我的 QQ 以便详细了解下事情的经过。

从对话中得知，姑娘叫小慧（化名），有一天她在逛某贴吧的时候，看到一篇招募童星的帖子，和很多人一样，小慧也有一个明星梦。看到后很心动就加了所谓星探的 QQ，直到发生了后面一些后悔莫及的事情才大梦初醒。

随后，我点进了小慧发来的地址，内容如下：

贴吧账号暂时没发现什么有用的信息，看来是个小号。除了发布一些招募童星的帖子之外，就只留下一个 QQ，目前看只能从这里下手了。

开始调查

为了避免引起怀疑，我去买了一个 QQ 号，伪装了一些动态（多数是一些追星、自拍、鸡汤之类的说说，好让骗子以为这是一个傻白甜），没过多久，骗子就通过了我的请求。

我试图通过骗子的 QQ 来获取一些必要的信息，但最终无果。等级很低，空间发现不了任何有用的信息，网络上与其相关的也是些招募童星的帖子，由此推测是小号。

不甘心，继续尝试获取 QQ 绑定的手机号码，发现其绑定的为 170 号段的虚拟号码，也就放弃了追查该手机号的想法（大多数虚拟号码都是未经过实名认证的），很显然，对方的防范意识很强。

嗯，意料之中，这样才有挑战。

接下来只能通过聊天来套取一些蛛丝马迹了，首先问了对方的公司，对方发来了营业执照，北京XXX文化传媒公司，但公司名在天眼查上找不到任何信息，证件是 PS 的，怕打草惊蛇也不好继续问下去。

随后我问了对方一些招募童星的条件以混淆试听，期间找了些自拍网图发过去（当然，是重新P过的，免得被对方通过图片的 Exif 发现了破绽）。

中间过程不表，直到来到了骗子最期待的视频验证，我以摄像头坏了为借口要求先进行语音验证（为后续发送视频文件捆绑木马做铺垫），并在验证过程中答应视频后续会以文件的形式发过去。感谢变声器，除了没看到本人之外，骗子对声音还是很满意的。

随后我随机填充了一个 50mb 大小的文件，后缀命名为 mp4，毫无疑问，是播放不了的。我将已经免杀后的木马名改为“如播放不了请点击安装插件.exe”。（这里插个题外话，我对远控的理解还停留在冰河、灰鸽子时代，所以这次要感谢我朋友的帮忙）

我将文件放到自己做的网站上（页面样式是克隆的网盘），至于为什么要这么大费周折而不是直接传输文件？自建网站的目的是为了获取到对方的 IP，防止木马程序被对方识破而一无所获，因为 QQ 早已经修复发送视频或者图片获取对方 IP 的漏洞。

接着，我已经做好的钓鱼网站丢了过去，随后开启了漫长的等待过程…

被发现了

然而两天时间过去了，并没有目标机器上线。逐询问，发现对方拉黑了！我！被发现了！

幸运的是，对方虽然没有运行木马，但点进去了网站，所以便成功的获取了对方的IP（120.0.XXX.XXX），通过查找发现，IP 地址显示为河北省石家庄市某小区。

但有个问题：无法确定具体地址，由于网络出口不同，通过 IP 定位会有或大或小的偏差（普遍为几公里到几十公里）。

柳暗花明

陷入了困境，范围还是太大，并且没有其他条件可以定位对方。不甘心，只好又把信息梳理一遍。这时突然注意到对方的贴吧资料：吧龄 2.4 年。等等，2.4 年！但是发布招募童星的帖子是最近一个月才开始的。直觉告诉我，账号在开始诈骗之前很有可能是对方的私人账号！

于是我查了一下贴吧绑定的手机号码，果然，绑定号码和 QQ 并不一致，不是虚拟号码！贴吧绑定号码为 159 开头尾号为 35，结合之前的 IP 定位数据，生成出几组号码其实不难，难就难在这么多号码要如何确认号码是对方的，哪怕再多知道 1 位尾号或者对方一个昵称、一个头像也行。

正当一筹莫展之际，突然想到之前有个重要线索我一直没有在意，那就是：小慧是先被骗了裸照，后被骗了钱。支付是最容易暴露一个人身份的环节。

之前没有在意是因为既然出来骗财骗色，那么身份证四件套是必须的吧？加上 QQ 绑定的是虚拟手机号码，所以主观认定支付账号肯定也是买来的。但上面的私人贴吧账号否定了我的想法，或许这个人的防范意识没有想象中那么强也不一定？

于是我联系到了小慧，让她把对方的收款账号发给我。对方的收款方式是支付宝，163 邮箱，随后我试图根据此邮箱来获取一些信息。

支付宝转账显示的名称为 X 珂。

通过 163 邮箱找回密码功能成功获取到了密保手机的尾号 135，和贴吧查出来的号码一致，密保邮箱也获取到一部分，为136XXXXX85@qq.com。

随后我将获取到的部分手机号码根据规则做了生成，我们已经知道对方手机号的前 3 位以及后 3 位，那么根据手机号段的生成规则是可以生成号码段的，最终筛选得到500多个手机号。

科普时间：手机号段组成为前3位是网络识别号、4-7位是地区编码、8-11位是用户号码(随机分配)。

将这些号码导入手机通讯录，通过 QQ 的“添加手机联系人”功能找到了十几个 QQ，根据支付宝头像以及 163 邮箱的密保邮箱比对成功锁定了目标（其实这里可以根据姓名 + 手机号在 XX 云市场找实名认证 API 查找的，奈何我穷，只好用这种笨方法…）。

我们先不着急加对方，接下来就要围绕 QQ 来做些文章。通过社工库查到了两条数据，然后惊喜的发现他竟然把姓和出生日期都写到了密码里(划重点啊朋友们，密码不要再包含个人信息了！泄露了可想改都改不了啊)！得来全不费工夫，当然这也少了后续的一些手段。

继续科普：社工库是黑客把泄露的数据归档而搭建的信息查询平台，主要包括密码、手机号码以及其他一些信息。

揪出元凶

接下来去验证一下，打开支付宝 - 转账 - 输入手机号，验证姓名，查了查百家姓，ying 字一共也就那么几个，很轻松的猜到了姓氏，验证成功！

接下来就是获取更详细的诸如地址之类的资料，有两种方法，第一种：既然前面已经知道了对方的省市县以及刚才查到的出生日期，那么根据身份证号码的生成规则，我们就可以推算出后四位从而进行验证。第二种：通过已有信息继续社工，找到微信、微博、具体地址等等信息。

思来想去之后，尝试继续搜索，得到了一条重要线索，是他高中时期在学校贴吧回复的帖子，这是第二个突破口，由此得到了对方的学校、以及同一届的同学信息。

好，我们汇总一下信息：

现在为止，我想信息已经足够多了，查不查的到详细地址也无所谓，于是我加了骗子的 QQ，于是有了下面的对话。

最终的结果是骗子答应删除照片并且退钱给小慧，后面小慧也告诉我她收到了骗子退回来的钱以及所谓的“道歉”。事情到这里，本篇文章也就告了一段落，当然文中提到的营珂（化名）只是其中一环一个小喽喽而已，在调查之中我发现招募童星是一个巨大的“市场”，甚至已经形成了招募童星、骗裸照、录制淫秽视频再从暗网贩卖的产业链。

调查总结

现在，我有些犹豫要不要顺着这条线查下去、查到真正的幕后黑手。一来，我已经暴露了自己的行踪，继续追查下去无疑难上加难；二来，这些人已经形成了完整的地下产业链。俗话说的好，夺人钱财如杀人父母，继续下去可能会引起一些不必要的麻烦，甚至自己的人身安全也会受到威胁。

思考良久，在正义感（好奇心）的驱动下，我决定继续查下去！不出所料，后续也确实遇到了很多麻烦，甚至差点被人发现了行踪，当然这些都是后话。

“温柔”在知乎发表文章称，在“童星吧”“招童星吧”等百度贴吧里，有大量招募童星的帖子，实际上是以招募童星的名义，通过要求检查身体素质的手段骗取未成年人裸照和裸体视频，并以此牟利。

网友张明关注童星招募骗局已久，在他看来，这是“各种欲望交织的结果”:孩子们渴望成名，却缺少正规渠道; 所谓“星探”骗取裸照，只为满足私欲;屏幕背后购买视频的人虽未直接行骗，却也推动着童星骗局的进行。

截止发文，百度童星吧依然充斥着各种招募童星的帖子、以及被骗后的控诉。我不知道为何招募童星骗局在18年就已经被曝光的情况下，百度贴吧还没有做监管？这是很难的事情么？我想不是。
